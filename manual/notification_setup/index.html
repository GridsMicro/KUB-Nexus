    🔔 Notification Setup (LINE Notify)

    ระบบ KUB-Nexus Autonomous Gateway รองรับการแจ้งเตือนแบบ Real-time ผ่าน LINE Notify เพื่อให้คุณไม่พลาดทุกความเคลื่อนไหวของการเทรดอัตโนมัติ ไม่ว่าจะเป็นสถานะการดำเนินงาน หรือผลกำไรที่เกิดขึ้น 🚀✨

    📋 ขั้นตอนการตั้งค่า LINE Notify

    กระบวนการนี้ง่ายมากและไม่จำเป็นต้องมีความรู้เรื่องโค้ดดิ้ง:

    1. เข้าสู่ระบบ LINE Notify:
    2. ไปที่เว็บไซต์ LINE Notify

    3. ล็อกอินด้วยบัญชี LINE ของคุณ

    4. สร้าง Token ใหม่:

    5. คลิกที่ชื่อบัญชีมุมขวาบน เลือกเมนู "My page" (หน้าของฉัน)
    6. เลื่อนลงมาล่างสุด คลิกปุ่ม "Generate token" (ออก Token)
    7. ตั้งชื่อการแจ้งเตือน: แนะนำให้ใช้ชื่อ [KUB-Nexus] เพื่อให้รู้ว่ามาจาก Gateway
    8. เลือกห้องแชท: เลือกให้ส่งเข้าแชทส่วนตัว (1-on-1) หรือสร้างกลุ่มใหม่เฉพาะกิจก็ได้

    9. คลิก "Generate" และ คัดลอก Token ที่ได้บนหน้าจอเอาไว้ให้ดี! (ระบบจะไม่แสดงให้เห็นอีก)

    10. นำ Token มาใส่ใน KUB-Nexus:

    11. ผ่านหน้าตั้งค่าตอน Boot เครื่อง (KUB-Nexus-Setup WiFi Portal) ให้กรอก LINE Token ลงในช่องที่กำหนด
    12. หมายเหตุ: หาก Token ไม่ถูกต้อง ระบบยังสามารถลุยโหมด Autonomous ได้ปกติ แต่การแจ้งเตือนจะไม่ทำงานเพื่อความลื่นไหลของระบบครับ

    💡 รูปแบบการแจ้งเตือนที่จะได้รับ

    เมื่อเกิดการซื้อขาย ระบบจะส่งข้อความที่อ่านง่าย สวยงาม และชัดเจน:

    ⚠️ ข้อมูลด้านความปลอดภัย (Data Privacy)

    • Local Storage Only: LINE Token ของคุณจะถูกบรรจุในหน่วยความจำ NVS (Non-Volatile Storage) ภายในบอร์ด ESP32 ไม่มีการอัปโหลดขึ้นเซิร์ฟเวอร์ใด ๆ นอกจากการส่งคำสั่งตรงไปที่ API ของ LINE
    • Fail-Safe Mechanism: หากเน็ตหลุดหรือไม่ได้กรอก Token ระบบหลักยังคงสั่งเทรดต่อไปได้อย่างอิสระ ไม่ทำให้จังหวะการทำกำไรสะดุดครับ 🌿🛡️
