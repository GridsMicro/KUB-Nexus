# 📖 คู่มือการใช้งาน KUB-Nexus (User Manual)

ยินดีต้อนรับสู่โลกของการสร้างความมั่งคั่งอย่างยั่งยืน คู่มือนี้จะช่วยให้คุณตั้งค่าและเริ่มต้นใช้งาน KUB-Nexus ได้อย่างรวดเร็วและปลอดภัย

---

## 🛠️ ขั้นตอนที่ 1: การเตรียม Bitkub API Key
คุณจำเป็นต้องมีสิทธิ์เข้าถึงข้อมูลบัญชีของคุณผ่านระบบ API ของ Bitkub ก่อน

1. เข้าสู่ระบบที่ [Bitkub.com](https://www.bitkub.com)
2. ไปที่เมนู **การจัดการ API** (API Management) หรือคลิก [ที่นี่](https://www.bitkub.com/th/api-management)
3. กดปุ่ม **สร้าง API Key ใหม่**
4. **การตั้งค่าสิทธิ์ (Permissions):** 
   - ✅ เลือก **ดูข้อมูล** (Read)
   - ✅ เลือก **ซื้อขาย** (Trading)
   - ❌ **ห้ามเลือก** การถอนเงิน (Withdraw) เพื่อความปลอดภัยสูงสุดของคุณ
5. จดจำ **API Key** และ **API Secret** ไว้ในที่ปลอดภัย (จะแสดงให้เห็นเพียงครั้งเดียว)

---

## 📱 ขั้นตอนที่ 2: การตั้งค่าผ่านหน้าเว็บ (สำหรับผู้ใช้ทั่วไป)
หากคุณใช้งานผ่านเบราว์เซอร์หรือแอปพลิเคชันของเรา:

1. เปิดแอปพลิเคชัน KUB-Nexus
2. ไปที่เมนู **ตั้งค่า (Settings)** รูปฟันเฟืองที่มุมขวาบน
3. กรอก **API Key** และ **API Secret** ในช่องที่กำหนด
4. ระบบจะทำการทดสอบการเชื่อมต่อ (Connection Test) หากสำเร็จคุณจะเห็นสถานะเป็นสีเขียว 🟢
5. ข้อมูลของคุณจะถูกเก็บรักษาไว้ด้วยการเข้ารหัสระดับธนาคาร (AES-256) ภายในเครื่องของคุณเท่านั้น

---

## 🤖 ขั้นตอนที่ 3: การตั้งค่าอุปกรณ์ IoT (ESP32) 
หากคุณมีเครื่อง KUB-Nexus Hardware วางบนโต๊ะ:

1. **เปิดเครื่อง:** เสียบสาย USB-C เข้ากับหัวชาร์จหรือคอมพิวเตอร์
2. **Setup Mode:** หากเปิดครั้งแรก ไฟที่เครื่องจะกระพริบเป็นสีฟ้า 🔵
3. **เชื่อมต่อ WiFi:** ใช้มือถือค้นหา WiFi ชื่อ `KUB-Nexus-Setup` และเชื่อมต่อ
4. **กรอกข้อมูล:** เมื่อหน้าเว็บตั้งค่าเด้งขึ้นมา ให้กรอก:
   - ชื่อ WiFi และรหัสผ่านที่บ้านของคุณ
   - Bitkub API Key ที่เตรียมไว้
5. **เสร็จสิ้น:** กด Save เครื่องจะรีบูตและเริ่มดึงราคา Bitcoin มาแสดงผลที่ไฟ RGB ทันที!

---

## 🌿 ขั้นตอนที่ 4: การใช้งานระบบ Eco-DCA
ระบบออมอัจฉริยะจะทำงานอัตโนมัติเมื่อ:

- คุณเปิดใช้งานโหมด **Autonomous Mode** ในหน้า Dashboard
- ระบบตรวจพบว่ามีการผลิตพลังงานสะอาด (Solar) ตามเงื่อนไขที่คุณตั้งไว้
- ราคา Bitcoin อยู่ในเกณฑ์ที่คุณต้องการ

---

## 🆘 การแก้ไขปัญหาเบื้องต้น
- **ไฟสีแดงกระพริบ 🔴:** การเชื่อมต่อ Bitkub API ผิดพลาด โปรดตรวจสอบ Key ของคุณ
- **ราคาไม่ขยับ:** ตรวจสอบสัญญาณอินเทอร์เน็ต
- **อยากลบข้อมูลออก:** ไปที่เมนู Settings และกดปุ่ม **Factory Reset** ข้อมูลทั้งหมดจะถูกลบออกจากเครื่องทันที
