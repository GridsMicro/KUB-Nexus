### **🔌 ผังการต่อวงจร (Wiring Diagram)**

การเชื่อมต่ออุปกรณ์ทั้งหมดเข้ากับ **ESP32** เพื่อประสิทธิภาพสูงสุดและรองรับการขยายตัวในอนาคต:

| อุปกรณ์ | ขาอุปกรณ์ | ขา ESP32 (GPIO) | บทบาทหน้าที่ |
| :--- | :--- | :--- | :--- |
| **OLED Display** | VCC / GND | 3.3V / GND | แหล่งจ่ายไฟ |
| | SCL / SDA | **GPIO 22 / GPIO 21** | สื่อสาร I2C (Price/Status) |
| **RGB LED** | GND (Long Pin) | GND | กราวด์ร่วม |
| | Red / Green / Blue | **GPIO 25 / GPIO 26 / GPIO 27** | สัญญาณสถานะ Failsafe |
| **Active Buzzer** | Positive (+) | **GPIO 32** | สัญญาณเตือนกำไร/ฉุกเฉิน |
| **Push Button** | Pin 1 / Pin 2 | **GPIO 4** / GND | ปุ่ม Setup/Reset (Pull-up) |

### **📐 ขั้นตอนการประกอบ (Assembly Steps)**

1. **การเตรียมบอร์ด:** ตรวจสอบความสมบูรณ์ของ ESP32 และทำการบัดกรี Header Pins (หากยังไม่ได้ดำเนินการ)
2. **การตั้งค่าระบบแสดงผล:** เชื่อมต่อจอ OLED เข้ากับขา I2C (21, 22) และทดสอบการแสดงผลเริ่มต้น
3. **ระบบความปลอดภัย (Failsafe LED):** เชื่อมต่อ RGB LED ผ่านตัวต้านทาน 220Ω เพื่อให้ระบบแจ้งสถานะการทำงานได้อย่างแม่นยำ
4. **การควบคุมภายนอก:** ติดตั้งปุ่มกดและ Buzzer เพื่อรองรับการโต้ตอบกับผู้ใช้ (Human-Machine Interface)
5. **การลงกล่อง (Enclosure):** บรรจุอุปกรณ์ทั้งหมดลงในเคสที่ออกแบบมาเพื่อการระบายความร้อนที่ดีเยี่ยม

---

### **⚠️ ข้อควรระวังด้านวิศวกรรม (Engineering Notes)**

* **Power Supply:** แนะนำให้ใช้แหล่งจ่ายไฟ DC 5V/2A ที่มีความเสถียรผ่านพอร์ต Micro-USB หรือ USB-C
* **I2C Pull-up:** ตรวจสอบว่าจอ OLED ของคุณมีตัวต้านทาน Pull-up ในตัวหรือไม่ (หากไม่มีอาจต้องเพิ่ม 4.7kΩ ที่ขา SDA/SCL)
* **Heat Management:** พื้นที่ภายในเคสควรมีช่องให้อากาศไหลเวียนเพื่อยืดอายุการใช้งานของชิป ESP32 ในการรันระบบ 24/7
